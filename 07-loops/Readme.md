# Loops

## Overview (EN)
Loops repeat commands. Bash supports `for`, `while`, `until`, and `case` (selection).

## အကျဉ်းချုပ် (MM)
Loop တွေက command တွေကို ထပ်ခါထပ်ခါ chạy ပေးပါတယ်။ Bash မှာ `for`, `while`, `until`, နဲ့ `case` ကို သုံးနိုင်ပါတယ်။

## Step by Step (EN)
1. Run `07-loops/01-for-loop.sh`.
2. Run `07-loops/02-while-loop.sh` and `07-loops/03-until-loop.sh`.
3. Try `07-loops/04-case-statement.sh`, then `break` and `continue` examples.

## Step by Step (MM)
1. `07-loops/01-for-loop.sh` ကို chạy ပါ။
2. `07-loops/02-while-loop.sh` နဲ့ `07-loops/03-until-loop.sh` ကို စမ်းပါ။
3. `07-loops/04-case-statement.sh` ကို စမ်းပြီး `break` နဲ့ `continue` ကို ကြည့်ပါ။

## Files
- `07-loops/01-for-loop.sh` For loop.
- `07-loops/02-while-loop.sh` While loop.
- `07-loops/03-until-loop.sh` Until loop.
- `07-loops/04-case-statement.sh` Case statement.
- `07-loops/05-break.sh` Break example.
- `07-loops/06-continue.sh` Continue example.
