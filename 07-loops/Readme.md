# Loops

## Overview (EN)
Loops repeat commands. Bash supports `for`, `while`, `until`, and `case` (selection).


## Step by Step (EN)
1. Run `07-loops/01-for-loop.sh`.
2. Run `07-loops/02-while-loop.sh` and `07-loops/03-until-loop.sh`.
3. Try `07-loops/04-case-statement.sh`, then `break` and `continue` examples.
