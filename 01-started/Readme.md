# Getting Started

## Overview (EN)
This chapter introduces what a shell is, how scripts run, and basics like PATH, quoting, tokenizing, and globbing.

## အကျဉ်းချုပ် (MM)
ဒီအခန်းမှာ shell ဆိုတာဘာလဲ၊ script ကို ဘယ်လို chạy လဲ၊ PATH, quoting, tokenizing, globbing အခြေခံတွေကို မိတ်ဆက်ထားပါတယ်။

## Step by Step (EN)
1. Run `01-started/01-shell.sh` to learn shell basics.
2. Run `01-started/02-hello-world.sh` for a simple script.
3. Read notes in `01-started/03-path-add.sh` about PATH and links.

## Step by Step (MM)
1. `01-started/01-shell.sh` နဲ့ shell အခြေခံကို လေ့လာပါ။
2. `01-started/02-hello-world.sh` ကို chạy ပြီး အလွယ်ကူဆုံး script ကို စမ်းပါ။
3. `01-started/03-path-add.sh` မှာ PATH နဲ့ link အကြောင်း ဖတ်ပါ။

## Files
- `01-started/01-shell.sh` Shell overview.
- `01-started/02-hello-world.sh` Hello world.
- `01-started/03-path-add.sh` PATH notes and links.
- `01-started/04-tilde-expansion.sh` `~` expansion.
- `01-started/05-qouting.sh` Quoting examples.
- `01-started/06-tokening.sh` Tokenizing example.
- `01-started/07-globbing.sh` Globbing examples.
