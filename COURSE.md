# Bash Master Course - Step by Step

## How To Use (EN)
1. Read the `Readme.md` in each chapter.
2. Run the scripts in order.
3. Change values and re-run to see how output changes.


## 01-started
1. Start with `01-started/01-shell.sh` to understand what a shell does.
2. Run `01-started/02-hello-world.sh` for your first script.
3. Follow the notes in `01-started/03-path-add.sh` to learn PATH and links.


## 02-variables
1. Run `02-variables/01-variables.sh` to learn variable basics.
2. Try environment variables with `02-variables/02-env-variables.sh`.
3. Use `02-variables/07-positional-parameter.sh` to pass arguments.


## 03-operators
1. Start with arithmetic in `03-operators/01-arithmetic-operator.sh`.
2. Compare values in `03-operators/02-comparison-operator.sh`.
3. Try file tests and redirection in `03-operators/05-file-testing-operator.sh` and `03-operators/06-redirection-operators.sh`.


## 04-data-types
1. Strings: `04-data-types/01-string.sh`.
2. Integers: `04-data-types/02-integer.sh`.
3. Floating-point: `04-data-types/03-floating.sh`.

1. Strings: `04-data-types/01-string.sh`.
2. Integers: `04-data-types/02-integer.sh`.
3. Floating-point: `04-data-types/03-floating.sh`.

## 05-arrays
1. Indexed arrays in `05-arrays/01-array.sh`.
2. Associative arrays in `05-arrays/02-associative-array.sh`.
3. Edit the arrays and re-run.


## 06-flow-control
1. If conditions in `06-flow-control/01-if.sh`.
2. Else and elif in `06-flow-control/02-if-else.sh` and `06-flow-control/03-elif.sh`.
3. String comparison in `06-flow-control/05-string-compasion.sh`.


## 07-loops
1. For loop: `07-loops/01-for-loop.sh`.
2. While and until: `07-loops/02-while-loop.sh`, `07-loops/03-until-loop.sh`.
3. Case, break, continue: `07-loops/04-case-statement.sh`, `07-loops/05-break.sh`, `07-loops/06-continue.sh`.


## 08-shell-functions
1. Read `08-shell-functions/Readme.md`.
2. Run `08-shell-functions/01-function.sh`.
3. Change the function argument and re-run.


## 09-file-testing
1. Read `09-file-testing/Readme.md`.
2. Practice the examples in your terminal.
3. Create a sample file and test it.


## 10-pipelines
1. Read `10-pipelines/Readme.md`.
2. Run the pipeline examples.
3. Replace commands to see different results.


## 11-process-substitution
1. Read `11-process-substitution/Readme.md`.
2. Try each example in terminal.
3. Replace commands to learn behavior.


## 12-regular-expressions
1. Read `12-regular-expressions/Readme.md`.
2. Try patterns with `grep`.
3. Edit patterns and test again.


## 13-bash-trap-debugging
1. Read `13-bash-trap-debugging/Readme.md`.
2. Try the trap examples.
3. Use `set -x` to see debugging output.
