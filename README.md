# Bash Master Course

## Overview (EN)
This repository is a step-by-step Bash course with short scripts and clear examples. Each chapter has a `Readme.md` and runnable `.sh` files.


## Quick Start (EN)
1. Open a chapter folder, for example `01-started`.
2. Read the notes in `01-started/Readme.md` if it exists.
3. Run a script with `bash 01-started/02-hello-world.sh`.
