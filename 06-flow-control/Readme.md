# Flow Control

## Overview (EN)
Flow control decides which commands run based on conditions. This chapter focuses on `if`, `elif`, and `else`.

## အကျဉ်းချုပ် (MM)
Flow control က condition အပေါ်မူတည်ပြီး command တွေကို run လုပ်မလား မလုပ်မလား ဆုံးဖြတ်ပေးပါတယ်။ ဒီအခန်းမှာ `if`, `elif`, `else` ကို လေ့လာပါမယ်။

## Step by Step (EN)
1. Run `06-flow-control/01-if.sh` to see a simple `if`.
2. Run `06-flow-control/02-if-else.sh` and `06-flow-control/03-elif.sh`.
3. Try string comparison in `06-flow-control/05-string-compasion.sh`.

## Step by Step (MM)
1. `06-flow-control/01-if.sh` နဲ့ simple `if` ကို ကြည့်ပါ။
2. `06-flow-control/02-if-else.sh` နဲ့ `06-flow-control/03-elif.sh` ကို စမ်းပါ။
3. `06-flow-control/05-string-compasion.sh` နဲ့ string comparison ကို စမ်းပါ။

## Files
- `06-flow-control/01-if.sh` Simple if.
- `06-flow-control/02-if-else.sh` If-else.
- `06-flow-control/03-elif.sh` If-elif-else.
- `06-flow-control/04-if-elif-else.sh` Another if-elif-else example.
- `06-flow-control/05-string-compasion.sh` String comparison.
