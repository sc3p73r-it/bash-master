# Bash Master Course - Step by Step

## How To Use (EN)
1. Read the `Readme.md` in each chapter.
2. Run the scripts in order.
3. Change values and re-run to see how output changes.

## ဘယ်လိုသုံးမလဲ (MM)
1. အခန်းတိုင်းက `Readme.md` ကိုဖတ်ပါ။
2. Script တွေကို အစဉ်လိုက် chạy ပါ။
3. တန်ဖိုးတွေကို ပြောင်းပြီး ပြန် chạy ကြည့်ပါ။

## 01-started
1. Start with `01-started/01-shell.sh` to understand what a shell does.
2. Run `01-started/02-hello-world.sh` for your first script.
3. Follow the notes in `01-started/03-path-add.sh` to learn PATH and links.

1. `01-started/01-shell.sh` နဲ့ shell ရဲ့ အလုပ်လုပ်ပုံကို နားလည်ပါ။
2. `01-started/02-hello-world.sh` ကို chạy ပြီး ပထမ script ကို စမ်းပါ။
3. `01-started/03-path-add.sh` မှာ PATH နဲ့ link အကြောင်းကို ဖတ်ပါ။

## 02-variables
1. Run `02-variables/01-variables.sh` to learn variable basics.
2. Try environment variables with `02-variables/02-env-variables.sh`.
3. Use `02-variables/07-positional-parameter.sh` to pass arguments.

1. `02-variables/01-variables.sh` နဲ့ variable အခြေခံကို လေ့လာပါ။
2. `02-variables/02-env-variables.sh` နဲ့ environment variables ကို စမ်းပါ။
3. `02-variables/07-positional-parameter.sh` နဲ့ arguments ပို့တာကို စမ်းပါ။

## 03-operators
1. Start with arithmetic in `03-operators/01-arithmetic-operator.sh`.
2. Compare values in `03-operators/02-comparison-operator.sh`.
3. Try file tests and redirection in `03-operators/05-file-testing-operator.sh` and `03-operators/06-redirection-operators.sh`.

1. `03-operators/01-arithmetic-operator.sh` နဲ့ arithmetic ကို စပါ။
2. `03-operators/02-comparison-operator.sh` နဲ့ comparison ကို စမ်းပါ။
3. `03-operators/05-file-testing-operator.sh` နဲ့ file test, `03-operators/06-redirection-operators.sh` နဲ့ redirection ကို စမ်းပါ။
