# Getting Started

## Overview (EN)
This chapter introduces what a shell is, how scripts run, and basics like PATH, quoting, tokenizing, and globbing.


## Step by Step (EN)
1. Run `01-started/01-shell.sh` to learn shell basics.
2. Run `01-started/02-hello-world.sh` for a simple script.
3. Read notes in `01-started/03-path-add.sh` about PATH and links.
