# Bash Master Course

## Overview (EN)
This repository is a step-by-step Bash course with short scripts and clear examples. Each chapter has a `Readme.md` and runnable `.sh` files.

## အကျဉ်းချုပ် (MM)
ဒီ repository က Bash ကို နားလည်လွယ်အောင် step-by-step နဲ့ သင်နိုင်ဖို့ စီစဉ်ထားတဲ့ course ဖြစ်ပါတယ်။ အခန်းတိုင်းမှာ `Readme.md` နဲ့ `.sh` ဖိုင်များ ပါဝင်ပါတယ်။

## Quick Start (EN)
1. Open a chapter folder, for example `01-started`.
2. Read the notes in `01-started/Readme.md` if it exists.
3. Run a script with `bash 01-started/02-hello-world.sh`.

## စပြီးလုပ်ရန် (MM)
1. အခန်းတစ်ခန်းကို ဖွင့်ပါ၊ ဥပမာ `01-started`.
2. `Readme.md` ရှိရင် အရင်ဖတ်ပါ။
3. `bash 01-started/02-hello-world.sh` နဲ့ script ကို chạy ပါ။

## Course Map (EN)
1. `01-started` - Shell basics, paths, quoting, tokenizing, globbing.
2. `02-variables` - Variables, environment, arrays, parameters.
3. `03-operators` - Arithmetic, comparison, logical, string, file tests, redirection, brace expansion.
4. `04-data-types` - Strings, integers, floating-point, boolean conventions.
5. `05-arrays` - Indexed arrays and associative arrays.
6. `06-flow-control` - If/elif/else and string comparisons.
7. `07-loops` - for/while/until, case, break/continue.
8. `08-shell-functions` - Function definition and parameters.
9. `09-file-testing` - File testing notes (readme only).
10. `10-pipelines` - Pipelines notes (readme only).
11. `11-process-substitution` - Process substitution notes (readme only).
12. `12-regular-expressions` - Regex notes (readme only).
13. `13-bash-trap-debugging` - Trap and debugging notes (readme only).
