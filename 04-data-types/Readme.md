# Data Types in Bash

## Overview (EN)
Bash treats everything as strings, but you can still do integer arithmetic and float calculations using tools like `bc` or `awk`.

## အကျဉ်းချုပ် (MM)
Bash မှာ data တွေကို string အဖြစ်သတ်မှတ်ပါတယ်။ Integer calculation တွေကို လုပ်နိုင်ပြီး float အတွက် `bc` သို့ `awk` ကို သုံးနိုင်ပါတယ်။

## Step by Step (EN)
1. Run `04-data-types/01-string.sh` to see string usage.
2. Run `04-data-types/02-integer.sh` to see integer arithmetic.
3. Run `04-data-types/03-floating.sh` to see float calculations.

## Step by Step (MM)
1. `04-data-types/01-string.sh` နဲ့ string ကို ကြည့်ပါ။
2. `04-data-types/02-integer.sh` နဲ့ integer arithmetic ကို စမ်းပါ။
3. `04-data-types/03-floating.sh` နဲ့ float calculation ကို စမ်းပါ။

## Files
- `04-data-types/01-string.sh` String example.
