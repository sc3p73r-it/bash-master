# Arrays

## Overview (EN)
Bash supports indexed arrays and associative arrays (Bash 4+). Indexed arrays use numeric indices, associative arrays use string keys.

## အကျဉ်းချုပ် (MM)
Bash မှာ indexed array နဲ့ associative array (Bash 4+) ကို သုံးနိုင်ပါတယ်။ Indexed array က index အလိုက်သုံးပြီး associative array က key-value အလိုက်သုံးပါတယ်။

## Step by Step (EN)
1. Run `05-arrays/01-array.sh` to learn indexed arrays.
2. Run `05-arrays/02-associative-array.sh` to learn associative arrays.
3. Add new elements and re-run.

## Step by Step (MM)
1. `05-arrays/01-array.sh` နဲ့ indexed array ကို စမ်းပါ။
2. `05-arrays/02-associative-array.sh` နဲ့ associative array ကို စမ်းပါ။
3. Element အသစ်တွေ ထည့်ပြီး ပြန် chạy ကြည့်ပါ။

## Files
- `05-arrays/01-array.sh` Indexed arrays.
- `05-arrays/02-associative-array.sh` Associative arrays.
